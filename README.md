# Bhakti Kajalkar — Developer Portfolio

A responsive personal portfolio website built using **HTML5** and **CSS3** as part of my internship project. The website showcases my skills, projects, and professional profile while following modern web development and accessibility practices.

## Overview

This project was developed during my internship to demonstrate frontend development skills without using frameworks or JavaScript libraries. The focus was on creating a clean, responsive, and user-friendly portfolio using semantic HTML and modern CSS techniques.

## Features

* Responsive design for mobile, tablet, and desktop
* Pure HTML5 and CSS3 implementation
* Semantic and accessible markup
* CSS-only mobile navigation
* Smooth hover effects and transitions
* Reusable and maintainable CSS structure
* Performance-focused design

## Tech Stack

* HTML5
* CSS3
* Flexbox
* CSS Grid
* CSS Variables
* Google Fonts
* SVG Icons
